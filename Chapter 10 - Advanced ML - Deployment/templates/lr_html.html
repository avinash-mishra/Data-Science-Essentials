Serving::Regression Model

This server provides endpoints to access a linear regression model

Endpoints

  • /train_model : endpoint to train a lr model
  • /predict_outcome : endpoint to get predictions from lr model
      • Parameters required:
    • house_size: the size of house required to make prediction for
    • reponse_type (optional, default is html): allowed value is json

Examples

  • Train a model : http://localhost:9090/train_model
  • Get Predictions as json : http://localhost:9090/predict_outcome?house_size=1750&response_type=json
  • Get Predictions as html : http://localhost:9090/predict_outcome?house_size=1750&response_type=html
